# Installation Guide

Learn how to install Avro in different environments.

## Prerequisites

- Node.js 16.0 or higher
- npm or yarn

## Installation Steps

### 1. Using npm

```bash
npm install avro
```

### 2. Using yarn

```bash
yarn add avro
```

## Verify Installation

To verify that Avro is installed correctly:

```bash
npm list avro
```

You should see the version number if the installation was successful.
